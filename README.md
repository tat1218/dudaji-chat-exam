## Quick Start

<details open>
<summary>Install</summary>

```bash
$ git clone https://github.com/tat1218/dudaji-chat-exam.git
$ cd dudaji-chat-exam
$ pip install -r requirements.txt
```

</details>

<details open>
<summary>Run Server</summary>

```bash
$ python server.py
```

</details>

<details open>
<summary>Run Client</summary>

```bash
$ python client.py --name ($your_name)
```

</details>
